# google_translate

A new Flutter project.

## Google_trans_simple

一个仿照Google translate UI的翻译App,目前有翻译单词的功能，支持10+语种，和收藏单词到本地

#### 技术栈:
lang: dart
数据持久化： floor库
全局数据：provider
api:百度翻译Api
